# Get source files used to build xcode projects

### install
`npm install xc-sources`

### usage
```Usage: xc-sources --project [project] --scheme [scheme] --arch [arch]

Options:
  --help     Show help                                                 [boolean]
  --version  Show version number                                       [boolean]
  --project                                                           [required]
  --scheme                                                            [required]
  --arch                                                      [default: "arm64"]
  ```
